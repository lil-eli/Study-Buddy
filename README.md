# Study-Buddy

App Name: StudyBuddy
App Type: Study Planner

Problem It Solves:
Helps students organize assignments and deadlines in one place. It is designed for students who need a simple way to keep track of their work.

Page 1 — Home Page (index.html):
This page introduces the StudyBuddy app and provides basic navigation to the other sections. It includes a short description of the app and a navigation bar to access all pages.

Page 2 — Tasks Page (page-2.html):
This page displays a list of current study tasks or assignments. It allows users to visually see what they need to complete in an organized list format.

Page 3 — Add Task Page (page-3.html):
This page contains a simple form where users can type in new tasks they want to add to their study list. It simulates task creation for future functionality.

Key Interactive Feature 1:
A navigation bar that allows users to switch between Home, Tasks and Add Task pages.

Key Interactive Feature 2:
A task list that visually displays assignments in an organized format using HTML lists.

Key Interactive Feature 3:
A form input field where users can type a new task (future functionality will allow adding tasks dynamically).
